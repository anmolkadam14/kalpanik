Kalpanik is an immersive mythological guidance application that delivers
AI-powered storytelling inspired by ancient Indian scriptures such as the
Bhagavad Gita, Ramayana, and Shiva Purana.

## Vision
To provide calm, ethical, and human-centered guidance through narrated
mythological stories rather than chatbot-style answers.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Voice: Browser Text-to-Speech
- AI: Expandable (Gemini / OpenAI / Ollama)
